## Getting Started

### Prerequisites

You need to use the software and how to install them.
* Cocoapod
  ```sh
  $ sudo gem install cocoapods
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/lnthe54/DopodCinema.git
   ```
2. Install PodFile
   ```sh
   pod install
   ```
3. Open workspace in Xcode
